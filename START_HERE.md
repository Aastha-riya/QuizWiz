# 🚀 QuizWiz - Start Here

## What You Have
✅ Complete Java web application (QuizWiz)  
✅ All code files ready  
✅ MySQL Connector configured  
✅ Clean project structure  

## What You Need to Do (5 Steps)

### 1️⃣ Setup Database
```cmd
mysql -u root -p < database\schema.sql
```

### 2️⃣ Build Project
```cmd
build.bat
```

### 3️⃣ Deploy to Tomcat
Copy `webapp` to `C:\apache-tomcat-9.0.x\webapps\QuizWiz`

### 4️⃣ Start Tomcat
```cmd
C:\apache-tomcat-9.0.x\bin\startup.bat
```

### 5️⃣ Open Browser
`http://localhost:8080/QuizWiz/quiz`
